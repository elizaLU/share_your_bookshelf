Personal project for booksharing portal for friends and families.
When I heard about GraphQL I was very enthusiastic about the possibilities it gives for swift querrying and sharing data! Right now, after my intense coding bootcamp the time has come to learn new technologies that I hope you use in the future.
In this project, on the backend, I use:

1. GraphQL
2. exress-graphql
3. MongoDB instance on the mLab portal
4. graphiql for easy test querrying in the browser
5. lodash for easy reusable code in the project

Users of this portal can:

- save a list of books they own
- create and join several readers-clubs
- share their home libraries with their clubs or particular users
- onwers can mark books as "read", "unavailable", "available"
- members of clubs can mark books as "read" and ask owners to reserve books for them
- users can write reviews of the books they have read
